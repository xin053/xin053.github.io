Python官方文档摘记

发表于 2016-10-06   |   分类于 Python   |  

说明

python官方文档无疑是每个学习python的pythoner必看的，除了The Python Tutorial新手必看之外，最重要莫过于The Python Language Reference和The Python Standard Library
前者简单介绍了有关python的实现，后者则是python的标准库的详解，无疑是很好的学习资源，在此记下对于我日常而言使用比较多的库，当作索引，以便将来查阅。

python官方文档版本：3.5.2
持续更新

The Python Language Reference：

  • Lexical analysis 词法分析 简单介绍了python一些功能的实现原理
  • Data model 介绍了很多双下划线的私有属性 值得反复看
  • The import system 导入系统介绍

The Python Standard Library：

  • Built-in Functions 必须熟练掌握
  • Built-in Types 基础内容 要熟悉
  • Text Processing Services 文本处理服务
    • format — 字符串格式化
    • re — 正则表达式操作 re正则库使用详解

  • Data Types

    • datetime — 基础时间和日期模块
    • calendar — General calendar-related functions
    • collections — 提供代替内置列表元组字典的多种数据结构
      • ChainMap — 通过链表将多个map链在一起 查找时优先查找前面的
      • Counter — Dict subclass for counting hashable items
      • deque — 双端队列
      • defaultdict — 给字典的value添加默认数据类型
      • namedtuple — 命名元组
      • OrderedDict — 有序字典 Dictionary that remembers insertion order
      • UserDict — dict 的一个封装类 主要用来拷贝一个字典的数据
      • UserList
      • UserString
    • heapq — 利用堆排序算法排序模块
    • bisect — 利用二分法将元素插入到已排序列表后依然保持已排序
    • weakref — 弱引用 用来避免循环引用导致对象没有被GC回收
    • types — Define names for built-in types that aren’t directly accessible as a builtin
    • copy — 分浅拷贝(不拷贝子对象)与深拷贝(拷贝子对象)
    • pprint — Data pretty printer
    • reprlib — 可以限制字符串长度的repr
    • enum — 3.4出现的
  • Numeric and Mathematical Modules
    • math — 常用数学操作 例如绝对值 阶乘 sin(x) cos(x)等
    • decimal — 更精确的数据类型 注意对于float数据要先转成字符串
    • fractions — 分数 如果需要准确精度也需要导入decimal包
    • random — 产生随机数 可指定产生哪个范围的随机数也能指定返回整数 但是不能用在安全方面
    • statistics — 统计学函数 例如求均值 中值 众数 标准差和方差
  • Functional Programming Modules
    • itertools — 提供多种常用便利操作 组合等等 返回值为生成器或者迭代器
    • functools — 提供一些方法对函数进行封装 以及reduce()
    • operator — 各种常用的操作符都封装到了该模块中 例如加减乘除 大小比较等等
  • File and Directory Access
    • pathlib — 路径库 pathlib路径库使用详解
    • os.path — 同样是路径库 与pathlib不同的是后者上升到了对象级别
    • stat — 显示文件mode等信息 也就是linux中 ls -al 显示的那些内容
    • tempfile — 创建临时文件或临时目录 使用特定的函数可以使得创建的临时文件在关闭的时候自动删除
    • glob — 匹配文件后缀名 可以包含路径
    • fnmatch — 匹配文件名 不能包含路径
    • linecache — 获取文件指定行内容并缓存
    • shutil — 提供高层次的文件操作 包括文件复制删除修改权限等 也有linux下which命令的功能 也提供简单的文件打包和解包 shell util 的简称
  • Data Persistence
    • pickle — 序列化 可序列化到文件 网络流或数据库 注意序列化安全漏洞
    • copyreg — 设置使用pickle和copy模块序列化或拷贝对象时将执行的函数
    • sqlite3 — sqlite数据库
  • Data Compression and Archiving
    • zlib — gzip压缩
    • gzip — gzip压缩
    • bz2 — bz2压缩
    • zipfile — zip压缩
    • tarfile — tar打包
  • File Formats
    • csv — csv格式相关
    • configparser — ini配置文件读取 configparser配置文件解析器使用详解
  • Cryptographic Services
    • hashlib — 各种加密算法
  • Generic Operating System Services
    • os — 与系统相关的 os库常用方法使用介绍
    • io — 输入输出流
    • argparse — 命令行工具 click命令行工具使用详解
    • logging — 日志 python logging模块使用教程 Python logging模块详解
    • getpass — 像linux中输入密码一样不显示输入的密码
  • Concurrent Execution
    • 线程
  • Interprocess Communication and Networking
    • socket — 底层socket
    • ssl
    • asyncio
  • Internet Data Handling
    • json
    • mimetypes — Map filenames to MIME types
    • base64 — Base16, Base32, Base64, Base85 Data Encodings
  • Structured Markup Processing Tools
    • html和xml解析 掌握BeautifulSoup即可 BeautifulSoup html与xml解析库使用详解
  • Internet Protocols and Support 常用网络协议
  • Internationalization 国际化
  • Debugging and Profiling
    • timeit — Measure execution time of small code snippets
  • Python Runtime Services
    • sys — System-specific parameters and functions
    • warnings — Warning control
    • contextlib — Utilities for with-statement contexts
    • atexit — Exit handlers
  • Importing Modules
    • zipimport — Import modules from Zip archives
    • modulefinder — Find modules used by a script
    • importlib – The implementation of import
  • Python Language Services
    • py_compile — Compile Python source files
    • compileall — Byte-compile Python libraries
    • dis — Disassembler for Python bytecode
  • MS Windows Specific Services
    • winreg – Windows registry access

python3.6新增

  • Cryptographic Services
    • secrets — 生成安全的随机数，功能同os.urandom()
